# Исправление переменной окружения Privy App ID в Vercel

## Проблема
В Vercel установлена неверная переменная окружения `VITE_PRIVY_APP_ID` со значением `cm25q62mj00nks8j5lxk4qyly`, которое вызывает ошибку 400 "Invalid Privy app ID".

## Решение

### Вариант 1: Удалить переменную окружения (Рекомендуется)
1. Зайдите в настройки проекта на Vercel
2. Перейдите в раздел "Environment Variables"
3. Найдите переменную `VITE_PRIVY_APP_ID`
4. Удалите её полностью

Приложение будет использовать fallback значение `cme84q0og02aalc0bh9blzwa9`, которое работает корректно.

### Вариант 2: Исправить значение переменной
1. Зайдите в настройки проекта на Vercel
2. Перейдите в раздел "Environment Variables"
3. Найдите переменную `VITE_PRIVY_APP_ID`
4. Измените её значение на: `cme84q0og02aalc0bh9blzwa9`

## Проверка
После изменения переменной окружения:
1. Сделайте redeploy проекта в Vercel
2. Откройте консоль браузера
3. Должно отображаться: `Privy App ID: cme84q0og02aalc0bh9blzwa9`
4. Не должно быть ошибок 400 при загрузке

## Важно
Правильный рабочий App ID для этого проекта: `cme84q0og02aalc0bh9blzwa9`
НЕ используйте: `cm25q62mj00nks8j5lxk4qyly` (этот ID недействителен)
